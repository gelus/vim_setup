Ben's Vim setup
====

Ben's easy to use vim setup.

how to use
----

```
$ git clone https://github.com/gelus/vim_setup.git
$ cd vim_setup
$ chmod +x setup.sh
$ ./setup.sh
```

And then create a link between ~/.vim and the repository.

```
$ ln -s vim_setup ~/.vim
```
( Note: you must use the full path to vim setup )
